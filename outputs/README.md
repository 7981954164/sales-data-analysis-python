# Outputs

This folder receives summary tables and reports generated by running the analysis pipeline: data_quality_report.json, kpi_summary.json, monthly_sales_summary.csv, regional_summary.csv, category_summary.csv, top_10_customers.csv, and top_10_products.csv. These files are BI-tool ready and can be loaded directly into Power BI, Tableau, or Excel. Run python -m src.main to populate this folder.
