# Images

This folder receives chart images (PNG) generated by running the analysis pipeline, for example monthly_sales_trend.png, revenue_by_region.png, revenue_by_category.png, top_10_customers.png, top_10_products.png, order_value_distribution.png, category_share_pie.png, and quantity_vs_profit_scatter.png. Run python -m src.main to populate this folder.
